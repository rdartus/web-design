À propos

Projet

Le site web présenté reprend les consignes données pour le cours de Web-Design, à savoir une implémentation de fonctionnalités Angularjs au travers d'un site design.

N'hésitez à explorer chaque recoin du site pour découvrir les nombreuses fonctionnalités, allant du comparateur aux filtres avancés, en passant par la navigation entre les Pokémons.

Le site correspond à un Pokédex géant, regroupant pour le moment tous les Pokémons de la 1ère génération. Des avis seront ajoutés au fur et à mesure afin de faire évoluer le site. Le site reprend les critères responsive comme demandés, en sachant que l'outil permettant de tester l'aspect responsive sur Chrome ne marche pas correctement. Néanmoins, cela fonctionne parfaitement sur Firefox.

Organisation

L'équipe s'est réunie de nombreuses fois pour avancer de façon commune. De plus, un outil collaboratif de type git a été mis en place pour faciliter le travail. Chaque membre a apporté son expertise sur le domaine, que ce soit en CSS, Angular ou HTML.
